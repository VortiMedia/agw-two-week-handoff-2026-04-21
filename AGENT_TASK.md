## Step 3: Build All Page Sections

- **Nav** — simple, premium header with phone-first CTA, service nav, and estimate action. No clutter.
- **Hero+Phone** — headline should feel heritage + trust + premium results, not generic contractor fluff. Push “Since 1906,” Fairfield/Westchester credibility, and fast estimate booking.
- **Services** — residential painting, commercial painting, cabinet refinishing, wallpaper removal/installation, concrete floor coatings, carpentry/light repair support if supported by current offer.
- **WhyUs** — 100+ year heritage, full-time background-checked crews, warranty, premium products, clean job sites, real project management.
- **Process** — dead-simple 3-step estimate flow: reach out, on-site estimate, job done right. Make booking feel frictionless.
- **Reviews** — testimonial density + local proof + review snippets that feel human. Prioritize trust over flashy motion.
- **ServiceArea** — Westchester County, Fairfield County, Rockland County, Putnam County. Make this map/list feel substantial and local.
- **FAQ** — pricing/estimate timing, warranty, insurance, scheduling, prep/cleanup, commercial vs residential, service area questions.
- **EmergencyCTA** — final high-intent section: get estimate, call now, or request a walkthrough. Not “emergency” language — use estimate/consultation framing.
- **Footer** — service areas, contact info, trust badges, business longevity, fast links, and clean secondary CTA.

**Rules for every component:**
- Mobile-first (375px)
- Uses CSS variables for all colors (never hardcode hex)
- Uses display font var for headings, body font for paragraphs
- 44px minimum tap targets
- All 8 interaction states (default/hover/focus/active/disabled/loading/error/success)
- No AI tells (see design-system/MASTER.md Anti-Patterns)
- prefers-reduced-motion respected for all animations
