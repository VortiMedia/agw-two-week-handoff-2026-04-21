# Selected Photos (AGW)

This is the **tasteful, site-focused** set derived from `assets/agw/photos/curated/high/`.

Everything in here is a **symlink** to a curated symlink (so originals stay untouched).

## Use
- **Hero**: `assets/agw/photos/selected/hero/`
- **Services**: `assets/agw/photos/selected/services/`
- **Process**: `assets/agw/photos/selected/process/`
- **Gallery**: `assets/agw/photos/selected/gallery/`
- **Detail** (trim, wainscot, etc): `assets/agw/photos/selected/detail/`
- **Team/Vehicle**: `assets/agw/photos/selected/team-vehicle/`
- **Stock**: `assets/agw/photos/selected/stock/` (clearly separated)

## Index
`assets/agw/photos/selected/INDEX.csv` includes:
- which set the image landed in
- tags + aspect ratio
- original path mapping
