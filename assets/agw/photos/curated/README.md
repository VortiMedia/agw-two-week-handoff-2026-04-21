# Curated Photos (AGW)

This folder is **symlinks** to the originals in `assets/agw/photos/raw/`.

## Buckets
- `high/` — recommended for the website (higher-res / heavier files)
- `low/` — usable only if needed (typically too small / UI screenshots / compression artifacts)
- `review-needed/` — risky keywords (don’t ship without approval)

## Index
`assets/agw/photos/curated/INDEX.csv` maps each curated filename back to its original file path + dimensions.

## Naming
`<category>--<slug>--<width>x<height>--<hash8>.<ext>`
